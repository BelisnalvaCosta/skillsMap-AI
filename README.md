# SkillMap AI

<img width="580" height="352" alt="image" src="https://github.com/user-attachments/assets/f4c5549c-6e16-4464-b71d-6f872c2879ab" />

https://belisnalvacosta.github.io/skillsMap-AI/

Projeto desenvolvimento na IDE Visual Studio Code com a orientação do professor Mário Martins (DIO).

Sistema web para **mapear habilidades pessoais e gerar ideias de renda automaticamente**.

O projeto foi desenvolvido para ajudar estudantes, profissionais e empreendedores a **identificar suas competências e descobrir possíveis formas de monetizá-las**.

A aplicação realiza uma **análise automática das habilidades inseridas**, classificando-as em categorias e sugerindo oportunidades de negócio ou serviços baseados nelas.

---

# Demonstração

O sistema permite:

* adicionar habilidades pessoais
* analisar automaticamente as competências
* classificar habilidades por categoria
* gerar ideias de renda baseadas nas habilidades informadas

Exemplo:

Entrada do usuário:

HTML
JavaScript
Organização de arquivos
Comunicação

Saída gerada:

Criação de sites e automações
Serviço de organização digital para empresas
Criação de cursos ou mentorias

# Como executar o projeto

1. Clone o repositório

```
git clone https://github.com/seu-usuario/skillmap-ai.git
```

2. Acesse a pasta do projeto

```
cd skillmap-ai
```

3. Abra o arquivo no navegador

```
index.html
```

Não é necessário instalar dependências ou rodar servidor.

---

# Como usar

1. Digite uma habilidade no campo de entrada
2. Clique em **Adicionar**
3. Repita para todas as suas habilidades
4. Clique em **Analisar Habilidades**
5. Clique em **Gerar Ideias de Renda**

O sistema irá sugerir formas de monetizar suas habilidades.
